UI Middle section:

Button1 calls the API from TextField2 with Authentication Bearer Token from TextField2. The Request Body looks like this:

{

"data": [

{

  "additionalProperty": null

}
],

"uniqueIdentifier": ""

}

WHERE the data array = the JSOn from MultiLineTextField1 and uniqueIdentifier = the value from TextField4.

{

"rowCount": 0,

}

Button2 then calls the API from TextField2 with the following appended to the URL "/{id}/query" where {id} = the value from reportID with Authentication Bearer Token from TextField2. The Request Body looks like this:

{

"prompt": ""

}

WHERE the prompt value is the value from MultiLineTextField2
